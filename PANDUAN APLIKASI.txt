üõí Sistem Admin Toko (Simple POS)

Aplikasi web sederhana untuk manajemen toko (Point of Sale) yang dibangun menggunakan Node.js, Express, dan MySQL. Aplikasi ini memungkinkan admin untuk mencatat transaksi pembelian, memonitor stok produk, dan membatalkan transaksi yang salah.

‚ú® Fitur Utama

Dashboard Interaktif: Ringkasan total pendapatan, jumlah transaksi, dan peringatan stok menipis.

Manajemen Transaksi: Input pembelian produk dengan perhitungan total harga otomatis.

Manajemen Stok Otomatis:

Stok berkurang saat transaksi berhasil.

Stok kembali bertambah (refund) saat transaksi dibatalkan.

Riwayat Transaksi: Melihat log penjualan lengkap dengan status (Sukses/Batal).

Sistem Autentikasi: Login aman menggunakan session untuk Admin.

Desain Modern: Antarmuka bersih menggunakan Material Design (Materialize CSS).

üõ†Ô∏è Teknologi yang Digunakan

Backend: Node.js, Express.js

Frontend: EJS (Templating Engine), Materialize CSS

Database: MySQL (via XAMPP)

Lainnya: express-session (Login), body-parser

Langkah 1: Persiapan Database

Buka XAMPP Control Panel, nyalakan modul Apache dan MySQL.

Buka browser ke http://localhost/phpmyadmin.

Klik tab SQL dan jalankan perintah berikut untuk membuat database dan akun admin default:

-- 1. Buat Database
CREATE DATABASE IF NOT EXISTS toko_admin_db;
USE toko_admin_db;

-- 2. Tabel Produk
CREATE TABLE IF NOT EXISTS products (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    price DECIMAL(10, 2) NOT NULL
);

-- 3. Tabel Stok
CREATE TABLE IF NOT EXISTS stocks (
    product_id INT PRIMARY KEY,
    quantity INT NOT NULL,
    FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
);

-- 4. Tabel Pembelian
CREATE TABLE IF NOT EXISTS purchases (
    id INT AUTO_INCREMENT PRIMARY KEY,
    product_id INT NOT NULL,
    quantity INT NOT NULL,
    total_price DECIMAL(15, 2) NOT NULL,
    status ENUM('completed', 'cancelled') DEFAULT 'completed',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (product_id) REFERENCES products(id)
);

-- 5. Tabel User (Admin)
CREATE TABLE IF NOT EXISTS users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    role ENUM('admin', 'staff') DEFAULT 'admin'
);

-- 6. Data Awal (Seeding)

üìñ Cara Menggunakan

Buka browser dan kunjungi http://localhost:3000.

Anda akan diarahkan ke halaman Login.

Masuk dengan akun default:

Username: admin

Password: admin123

Input Transaksi:

Pilih produk dari dropdown.

Masukkan jumlah.

Klik "Proses Transaksi". Stok akan otomatis berkurang.

Batalkan Transaksi:

Lihat tabel "Riwayat Transaksi" di sebelah kanan.

Klik tombol X (Merah) pada transaksi yang ingin dibatalkan.

Status akan berubah menjadi "Batal" dan stok produk akan dikembalikan otomatis.

üìÇ Struktur Database

users: Menyimpan data login admin.

products: Menyimpan nama dan harga barang.

stocks: Menyimpan jumlah stok barang (berelasi dengan products).

purchases: Menyimpan log transaksi penjualan.

‚ö†Ô∏è Troubleshooting

Error "Failed to lookup view": Pastikan file .ejs Anda berada di dalam folder bernama views.

Error Koneksi Database: Pastikan XAMPP MySQL sudah menyala dan tidak ada password pada user root (default XAMPP). Jika Anda menggunakan password database, edit konfigurasi di server.js.
